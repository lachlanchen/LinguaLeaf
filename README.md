# LingualLeaf

[![LazyingArt banner](https://github.com/lachlanchen/lachlanchen/raw/main/figs/banner.png)](https://github.com/lachlanchen/lachlanchen/blob/main/figs/banner.png)

LingualLeaf is the dedicated PDF artifact shelf for [PocketPolyglot](https://github.com/lachlanchen/PocketPolyglot). It stores compressed final multilingual pocket-book PDFs, first-page previews, and the export manifest, while PocketPolyglot keeps the source text, JSON, TeX, scripts, and generation workflow.

Website: [learn.lazying.art](https://learn.lazying.art)  
Source/tooling repository: [lachlanchen/PocketPolyglot](https://github.com/lachlanchen/PocketPolyglot)

## What Is Here

- 94 compressed final PDFs under `docs/pocketpolyglot/books/`.
- 41 first-page preview images under `assets/max-language-previews/`.
- Color and black-white editions for every exported book.
- Maximum available language families: `jp-zh`, `en-jp-zh`, and `wenyan-en-jp-zh`.
